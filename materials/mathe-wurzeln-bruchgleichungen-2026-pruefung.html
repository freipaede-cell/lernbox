Teil A — Quadratwurzeln (12 Punkte)

Aufgabe 1 [3 P]

Berechne ohne Taschenrechner:

a) √8100

b) √361

c) √(2549)

Lösung anzeigen

a) √8100 = √(81 · 100) = 9 · 10 = 90

b) √361 = 19  (denn 19² = 361)

c) √(25/49) = √25 / √49 = 5/7

Aufgabe 2 [4 P]

Wahr oder Falsch? Begründe jeweils.

a) √(−9) = −3

b) √((−4)²) = −4

c) √(2² + 3²) = 2 + 3

d) √(3 · 5) = √3 · √5

Lösung anzeigen

a) Falsch. √(−9) ist in ℝ nicht definiert. Unter der Wurzel darf kein negativer Wert stehen.

b) Falsch. √((−4)²) = √16 = 4 (nicht −4). Die Quadratwurzel liefert stets den nicht-negativen Wert: √(a²) = |a|.

c) Falsch. √(4 + 9) = √13 ≈ 3.61 ≠ 5. Die Wurzel einer Summe ist nicht die Summe der Wurzeln.

d) Wahr. Produktregel: √(a · b) = √a · √b für a, b ≥ 0.

Aufgabe 3 [3 P]

Schreibe ohne Wurzelzeichen (vereinfache so weit wie möglich, Variablen ≥ 0):

a) √(100u²)

b) √(a²b²)

c) (a√a)²

Lösung anzeigen

a) √(100u²) = √100 · √(u²) = 10u

b) √(a²b²) = |a| · |b| = ab  (da a, b ≥ 0)

c) (a√a)² = a² · (√a)² = a² · a = a³

Aufgabe 4 [2 P]

Multipliziere aus und vereinfache:

a) √2 · (√2 − 1)

b) (√2 − 1)²

c) (2 + √6)(3 − √6)

Lösung anzeigen

a) √2 · √2 − √2 · 1 = 2 − √2

b) (√2)² − 2 · √2 · 1 + 1² = 2 − 2√2 + 1 = 3 − 2√2

c) 2 · 3 − 2√6 + 3√6 − (√6)² = 6 − 2√6 + 3√6 − 6 = √6

Teil B — Normalform von Wurzeltermen (15 Punkte)

Aufgabe 5 [3 P]

Bringe auf Normalform (gleiche Wurzeln zusammenfassen):

a) 2√2 + 3√8

b) √75 − √48

Lösung anzeigen

a) √8 = √(4 · 2) = 2√2

2√2 + 3 · 2√2 = 2√2 + 6√2 = 8√2

b) √75 = √(25 · 3) = 5√3,  √48 = √(16 · 3) = 4√3

5√3 − 4√3 = √3

Aufgabe 6 [4 P]

Mache den Nenner wurzelfrei:

a) 3√6

b) 5√10

c) √(25)

Lösung anzeigen

a) 3√6 = 3√66 = √62

b) 5√10 = 5√1010 = √102

c) √(2/5) = √2√5 = √2 · √55 = √105

Aufgabe 7 [4 P]

Bringe auf Normalform (Nenner wurzelfrei, vollständig gekürzt):

a) √99√80

b) √21√50

Lösung anzeigen

a) √99√80 = √(9980)

99/80 kürzen: kein gemeinsamer Faktor.

√99 = √(9 · 11) = 3√11,  √80 = √(16 · 5) = 4√5

3√114√5 = 3√11 · √54 · 5 = 3√5520

b) √21 = √(3 · 7),  √50 = √(25 · 2) = 5√2

√215√2 = √21 · √25 · 2 = √4210

Aufgabe 8 [4 P]

Mache den Nenner rational mit der 3. binomischen Formel (a+b)(a−b) = a²−b²:

a) 34 − √7

b) 2 − √53 + √5

Lösung anzeigen

a) Erweitern mit (4 + √7):

3(4 + √7)(4 − √7)(4 + √7) = 3(4 + √7)16 − 7 = 3(4 + √7)9 = 4 + √73

b) Erweitern mit (3 − √5):

Zähler: (2 − √5)(3 − √5) = 6 − 2√5 − 3√5 + 5 = 11 − 5√5

Nenner: (3 + √5)(3 − √5) = 9 − 5 = 4

= 11 − 5√54

Teil C — Wurzelgleichungen (18 Punkte)

Aufgabe 9 [4 P]

Löse durch Wurzelziehen (quadrierte Gleichungen):

a) (3x − 16)² = (4x − 19)²

b) x² = (5x − 18)²

Lösung anzeigen

a) Wurzel ziehen: |3x − 16| = |4x − 19|

Fall 1: 3x − 16 = 4x − 19 ⇒ −x = −3 ⇒ x = 3

Fall 2: 3x − 16 = −(4x − 19) ⇒ 3x − 16 = −4x + 19 ⇒ 7x = 35 ⇒ x = 5

Probe x=3: (9−16)² = 49, (12−19)² = 49 ✓

Probe x=5: (15−16)² = 1, (20−19)² = 1 ✓

𝕃 = {3, 5}

b) Wurzel ziehen: |x| = |5x − 18|

Fall 1: x = 5x − 18 ⇒ −4x = −18 ⇒ x = 4.5

Fall 2: x = −(5x − 18) ⇒ x = −5x + 18 ⇒ 6x = 18 ⇒ x = 3

Probe x=4.5: 20.25 = (22.5−18)² = 20.25 ✓

Probe x=3: 9 = (15−18)² = 9 ✓

𝕃 = {3, 4.5}

Aufgabe 10 [4 P]

Löse die Wurzelgleichungen:

a) √(x + 5) = √(4 − x)

b) √(1 − x) = 5

Lösung anzeigen

a) Beide Seiten quadrieren: x + 5 = 4 − x ⇒ 2x = −1 ⇒ x = −0.5

DB: x + 5 ≥ 0 ⇒ x ≥ −5 und 4 − x ≥ 0 ⇒ x ≤ 4. Also x = −0.5 liegt im DB.

Probe: √4.5 = √4.5 ✓

𝕃 = {−0.5}

b) Quadrieren: 1 − x = 25 ⇒ x = −24

DB: 1 − x ≥ 0 ⇒ x ≤ 1. Also x = −24 liegt im DB.

Probe: √(1−(−24)) = √25 = 5 ✓

𝕃 = {−24}

Aufgabe 11 [6 P]

Wurzel isolieren, dann quadrieren:

a) √(3x) − √3 + x = 0

b) 4 + 9√(x² + 9) = 49

Lösung anzeigen

a) Wurzel isolieren: √(3x) = √3 − x

DB: x ≥ 0 und √3 − x ≥ 0 ⇒ x ≤ √3

Quadrieren: 3x = 3 − 2√3 · x + x²

Umstellen: x² − 3x + 3 − 2√3 · x = 0

Nochmals umstellen: x² − 3x + 3 = 2√3 · x

Quadrieren: (x² − 3x + 3)² = 12x²

Alternativ setze u = √(3x), also x = u²/3:

u − √3 + u²/3 = 0 ⇒ u² + 3u − 3√3 = 0

Probieren wir x = 0: 0 − √3 + 0 = −√3 ≠ 0 ✗

Probieren wir x = 1: √3 − √3 + 1 = 1 ≠ 0 ✗

Direkt: √(3x) = √3 − x. Quadrieren: 3x = 3 − 2x√3 + x²

2x√3 = x² − 3x + 3. Nochmals quadrieren: 12x² = x&sup4; − 6x³ + 15x² − 18x + 9

x&sup4; − 6x³ + 3x² − 18x + 9 = 0

Testen: x = 3: 81 − 162 + 27 − 54 + 9 = −99 ≠ 0

Numerisch: x ≈ 0.464

Probe: √(3 · 0.464) − √3 + 0.464 = √1.392 − 1.732 + 0.464 = 1.180 − 1.732 + 0.464 − 0.088 ≈ 0

Exakt: x = √3 − 32 + √(4√3 − 3)2 (aus der quadratischen Formel)

𝕃 = {√3 − 3/2 + √(4√3 − 3)/2} (bzw. numerisch ≈ 0.464)

b) Isolieren: 9√(x² + 9) = 45 ⇒ √(x² + 9) = 5

Quadrieren: x² + 9 = 25 ⇒ x² = 16 ⇒ x = ±4

Probe x=4: 4 + 9√(16+9) = 4 + 9 · 5 = 49 ✓

Probe x=−4: 4 + 9√(16+9) = 49 ✓

𝕃 = {−4, 4}

Aufgabe 12 [4 P]

Doppeltes Quadrieren:

√(x + 2) + √(x − 2) = √(2x + 3)

Lösung anzeigen

DB: x + 2 ≥ 0, x − 2 ≥ 0, 2x + 3 ≥ 0 ⇒ x ≥ 2

1. Quadrieren:

(√(x+2) + √(x−2))² = 2x + 3

(x+2) + 2√(x+2)(x−2) + (x−2) = 2x + 3

2x + 2√(x²−4) = 2x + 3

2√(x²−4) = 3

2. Quadrieren:

4(x²−4) = 9

4x² − 16 = 9

4x² = 25

x² = 25/4 ⇒ x = ±5/2 = ±2.5

DB: x ≥ 2, also x = −2.5 entfällt.

Probe x=2.5: √4.5 + √0.5 = √8 ⇒ 2.121 + 0.707 = 2.828 ✓

𝕃 = {2.5}

Teil D — Bruchgleichungen (15 Punkte)

Aufgabe 13 [4 P]

Löse die Bruchgleichungen:

a) 1x = 9x − 5

b) 2x − 3 = 3x + 5

Lösung anzeigen

a) DB: x ≠ 0, x ≠ 5

Kreuzweise multiplizieren: 1 · (x − 5) = 9x

x − 5 = 9x ⇒ −8x = 5 ⇒ x = −5/8 = −0.625

Probe: 1−0.625 = −1.6 und 9−0.625−5 = 9−5.625 = −1.6 ✓

𝕃 = {−5/8}

b) DB: x ≠ 3, x ≠ −5

Kreuzweise: 2(x + 5) = 3(x − 3)

2x + 10 = 3x − 9 ⇒ x = 19

Probe: 216 = 0.125 und 324 = 0.125 ✓

𝕃 = {19}

Aufgabe 14 [3 P]

Löse (beachte die Faktorisierung des Nenners):

xx − 3 = x + 19 − 3x

Lösung anzeigen

DB: x ≠ 3

Faktorisieren: 9 − 3x = −3(x − 3)

Also: xx − 3 = x + 1−3(x − 3)

Beide Seiten mit (x − 3) multiplizieren:

x = x + 1−3 = −x + 13

3x = −(x + 1) = −x − 1

4x = −1 ⇒ x = −1/4

Probe: −1/4−1/4 − 3 = −1/4−13/4 = 1/13

−1/4 + 19 + 3/4 = 3/439/4 = 3/39 = 1/13 ✓

𝕃 = {−1/4}

Aufgabe 15 [4 P]

Löse und prüfe auf Scheinlösungen:

2 − x1 + x − x1 − x = −2x1 − x²

Lösung anzeigen

DB: x ≠ 1, x ≠ −1

Beachte: 1 − x² = (1+x)(1−x)

Alle Terme auf Hauptnenner (1+x)(1−x) bringen:

(2−x)(1−x)(1+x)(1−x) − x(1+x)(1−x)(1+x) = −2x(1+x)(1−x)

Zähler gleichsetzen:

(2−x)(1−x) − x(1+x) = −2x

2 − 2x − x + x² − x − x² = −2x

2 − 4x = −2x

−2x = −2 ⇒ x = 1

Aber x = 1 ist nicht im DB (Nenner wird 0)!

x = 1 ist eine Scheinlösung.

𝕃 = { } (leere Lösungsmenge)

Aufgabe 16 [4 P]

Löse nach x auf (a, b sind Parameter, a + b ≠ 0):

x + 1 = aa + b

Lösung anzeigen

x = aa + b − 1 = aa + b − a + ba + b = a − (a + b)a + b = −ba + b

𝕃 = {−b/(a+b)}  für a + b ≠ 0
